SORTING

df.sort_values("Column", ascending=False)

--------------------------------------------------

df["Column"].min()

Returns the average.

df["Column"].sum()
Returns the sum.

df["Column"].count()
Returns the number of non-null values.

--------------------------------------------------

GROUP BY

df.groupby("Column")
Groups rows by a column.

df.groupby("Column")["Value"]

df.groupby("Column")["Value"].max()

df.groupby("Column")["Value"].min()
Returns the minimum value within each group.

df.groupby("Column")["Value"].mean()
Returns the average value within each group.

df.groupby("Column")["Value"].sum()
Returns the total value within each group.

MULTIPLE AGGREGATIONS

JOIN / MERGE

Combines two DataFrames using a common column.

Common join types:

Only matching rows.

pd.merge(df1, df2, on="ID", how="left")
All rows from left table.

pd.merge(df1, df2, on="ID", how="right")
All rows from right table.

pd.merge(df1, df2, on="ID", how="outer")
All rows from both tables.
